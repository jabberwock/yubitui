<objective>
Replace the hardcoded `available_keys[0]` import with an interactive key picker so users can choose which GPG key to import to their YubiKey.

Purpose: Currently pressing Enter on the ImportKey screen always imports the first key. Users with multiple GPG keys have no way to select which one to import.
Output: Interactive key selection UI with arrow-key navigation and visual highlight.
</objective>
