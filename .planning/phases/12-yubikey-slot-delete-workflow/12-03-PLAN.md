<task type="checkpoint:human-verify" gate="blocking">
  <what-built>
Complete slot delete workflow for both OpenPGP (Keys screen) and PIV (PIV screen):
- OpenPGP: Press D on a key slot -> enter Admin PIN -> confirm -> key material destroyed
- PIV: Press D on a PIV slot -> enter management key (hex, or Enter for default) -> confirm with firmware-aware messaging -> certificate (and key on 5.7+) deleted
  </what-built>
  <how-to-verify>
1. Start the app: `cargo run` (with YubiKey inserted) or `cargo run -- --mock` for mock mode

2. **OpenPGP delete flow (Keys screen):**
   - Navigate to Keys screen
   - Select a slot that has a key (SIG/ENC/AUT)
   - Press 'D'
   - Enter Admin PIN when prompted
   - Confirm the deletion
   - Verify success popup appears
   - Go back to Keys screen — the slot should now show as empty

3. **PIV delete flow (PIV screen):**
   - Navigate to PIV screen
   - Use Up/Down to select an occupied slot (9a/9c/9d/9e)
   - Press 'D'
   - Enter management key (or press Enter for default)
   - Confirm the deletion — note the firmware-aware message
   - Verify success popup with correct message (cert only vs cert+key)
   - Go back to PIV screen — the slot should now show as empty

4. **Edge cases to check:**
   - Press D on an empty slot — should show "No key/certificate to delete" popup
   - Enter wrong Admin PIN on OpenPGP — should show error with retry count
   - Enter wrong management key on PIV — should show auth failure message
   - On firmware < 5.7: confirm message should note key material cannot be removed

5. Verify no visual regressions on other screens (dashboard, help, etc.)
  </how-to-verify>
