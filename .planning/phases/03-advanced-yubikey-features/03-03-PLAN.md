<objective>
Wire multi-key support, touch policy display/setting, and attestation viewing into the TUI. This is the big integration plan that connects the backend modules from Plan 02 to the UI and evolves the App struct from single-key to multi-key.

The detection layer learns to enumerate YubiKeys via `ykman list --serials`. The App struct evolves from `yubikey_state: Option<YubiKeyState>` to `yubikey_states: Vec<YubiKeyState>` with a selected index. The existing `yubikey_state()` accessor returns `selected_yubikey()` for backward compatibility. Dashboard shows active key indicator with Tab switching. Keys screen displays touch policies per slot, allows setting touch policy (with irreversibility warnings), and shows attestation certificates in a popup.

Purpose: After this plan, power users can manage multiple keys, see touch/attestation status, and change touch policies — the core Phase 3 user-facing features.

Output: Working multi-key switcher, touch policy display and set, attestation popup in the TUI.
</objective>

        ```rust
        fn execute_touch_policy_set(&mut self, slot: &str, policy: &crate::yubikey::touch_policy::TouchPolicy) -> Result<()> {
            let serial = self.yubikey_state().map(|yk| yk.info.serial);

            // Drop to terminal for Admin PIN entry
            crossterm::terminal::disable_raw_mode()?;
            crossterm::execute!(std::io::stdout(), crossterm::terminal::LeaveAlternateScreen)?;

            let result = crate::yubikey::touch_policy::set_touch_policy(slot, policy, serial);
            match result {
                Ok(mut child) => {
                    let status = child.wait();
                    match status {
                        Ok(s) if s.success() => {
                            self.key_state.message = Some(format!("Touch policy set to {} for {}", policy, slot));
                        }
                        Ok(_) => {
                            self.key_state.message = Some("Touch policy change cancelled or failed".to_string());
                        }
                        Err(e) => {
                            self.key_state.message = Some(format!("Error: {}", e));
                        }
                    }
                }
                Err(e) => {
                    self.key_state.message = Some(format!("Error: {}", e));
                }
            }

            // Restore TUI
            crossterm::execute!(std::io::stdout(), crossterm::terminal::EnterAlternateScreen)?;
            crossterm::terminal::enable_raw_mode()?;

            // Refresh state
            self.yubikey_states = YubiKeyState::detect_all().unwrap_or_default();
            if self.selected_yubikey_idx >= self.yubikey_states.len() {
                self.selected_yubikey_idx = 0;
            }
            self.key_state.screen = ui::keys::KeyScreen::Main;
            Ok(())
        }
        ```
