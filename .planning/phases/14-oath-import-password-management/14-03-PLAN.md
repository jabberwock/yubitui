**SetOathPasswordScreen** (OATH-08 — set when no password exists):

```rust
#[derive(Clone, Copy, PartialEq)]
enum SetPasswordStep { NewPassword, Confirm }

pub struct SetOathPasswordScreen {
    step: Cell<SetPasswordStep>,
    new_password: RefCell<String>,
    input: RefCell<String>,
    error: RefCell<Option<String>>,
    own_id: Cell<Option<WidgetId>>,
}
```

Steps:
1. `NewPassword`: prompt "Enter new password:", masked input, Enter advances to Confirm (store in new_password, clear input)
2. `Confirm`: prompt "Confirm new password:", masked input, Enter checks match → calls `set_oath_password(&new_password)`

On success: pop self + push `PopupScreen::new("Password Set", "OATH password set. You will be prompted each session.")`
On mismatch: set error "Passwords do not match", back to NewPassword step
On model error: set error, stay on Confirm step

Key bindings: Esc=cancel (pop), Enter=next

`on_action("unlock")`:
1. Derive key: `let key = crate::model::oath::derive_oath_key(&input, &self.device_name)`
2. Connect to card in same session:
   - kill_scdaemon + sleep
   - Connect, SELECT, VALIDATE via `crate::model::oath::validate_oath_session(&card, &key, &device_challenge)`
   - On VALIDATE ok: call `crate::model::oath::get_oath_state()` to reload credentials
   - pop self + push fresh `OathScreen::new_with_key(Some(fresh_state))`
3. On error: set error message, clear input

Note: `OathUnlockScreen` is pushed from `OathScreen.on_action("refresh")` when `get_oath_state()` returns
`password_required = true` and `device_challenge` is non-empty. Update the "refresh" arm in
`OathScreen.on_action()`:

```rust
"refresh" => {
    let fresh = crate::model::oath::get_oath_state().ok();
    match &fresh {
        Some(s) if s.password_required && !s.device_challenge.is_empty() => {
            ctx.pop_screen_deferred();
            ctx.push_screen_deferred(Box::new(OathUnlockScreen::new(
                s.device_name.clone(),
                s.device_challenge.clone(),
            )));
        }
        _ => {
            ctx.pop_screen_deferred();
            ctx.push_screen_deferred(Box::new(OathScreen::new(fresh)));
        }
    }
}
```
