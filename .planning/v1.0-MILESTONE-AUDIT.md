## Tech Debt Summary

### Critical Items (should fix before v1.1)

1. **Phase 04 VERIFICATION.md missing** — `04-programmatic-subprocess-control` was never formally verified. Requirements NO-ESCAPE-01 and IN-TUI-FEEDBACK-01 are substantively met (cross-confirmed by Phase 05 verifier and integration checker) but lack documentation. Run `/gsd:validate-phase 04` to close this gap.

2. **`cargo fmt` failing** — `src/app.rs`, `src/ui/pin.rs`, `src/ui/widgets/popup.rs`, `src/utils/config.rs` have formatting diffs. CLAUDE.md requires this check to pass before commits. Run `cargo fmt` to fix.

### Minor Items (backlog)

3. **`Char('t')` stale message** — Touch policy navigation arm doesn't clear `key_state.message`. Cosmetic UX defect (operation executes correctly). One-line fix: add `self.key_state.message = None;` at `app.rs:286`.

4. **50ms sleep missing** — `kill_scdaemon()` in `connect_to_openpgp_card()` and `get_piv_state()` lacks post-kill delay. Works on macOS/tested hardware; potential robustness issue on Linux with slow scdaemon teardown.

5. **ROADMAP cosmetics fixed** — Checkboxes for 02-04 and 05-04/05/06 were `[ ]`; corrected to `[x]` in this audit.
